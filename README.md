# 姫路IT系勉強会（HISTUDY）行動規範

## 概要

このリポジトリは、姫路IT系勉強会（HISTUDY）コミュニティの行動規範（Code of Conduct）を管理しています。

HISTUDYは、技術知識の共有、スキルの向上、そして相互の成長を目的とした、すべての参加者にとって安全で歓迎的な環境を提供することにコミットしています。

**注）このリポジトリは姫路IT系勉強会の行動規範を検討し、議論するためのリポジトリです。そのため現時点では適用されていません。**

## 目的

- コミュニティ内でのハラスメント、差別、嫌がらせを防止する
- すべての参加者にとって包摂的で尊重される環境を実現する
- コミュニティメンバー間の信頼と協力を構築する

## 内容

- [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) - 姫路IT系勉強会の詳細な行動規範
